# Глава 2. Реализация плагина

2.1. Регистрация произвольных типов записей (product, news) и таксономий
2.2. Реализация мета-боксов для товаров (цена, артикул, остаток, вес, внешняя ссылка) и новостей (ссылка «читать далее»)
2.3. Реализация шорткода `[product_table]` и его атрибутов
2.4. REST API endpoint `GET /my-catalog/v1/product-table` с серверной обработкой DataTables (поиск, сортировка, фильтрация)
2.5. Страница настроек плагина (выбор отображаемых колонок таблицы)
2.6. Реализация динамического Gutenberg-блока «Карусель новостей» с ServerSideRender
2.7. Подключение frontend-библиотек (DataTables, Swiper) через CDN
2.8. Сборка JavaScript/CSS через `@wordpress/scripts`
2.9. Развёртывание и системные требования
